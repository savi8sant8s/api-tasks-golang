# Gotasks (v1)

Api de registro de tarefas construída em Golang (Gin e Gorm) utilizando o padrão MVC e melhores práticas conhecidas - por mim até o momento.

-----

Baixar dependências:
```
go get .
```
Rodar projeto:
```
go run .
```
